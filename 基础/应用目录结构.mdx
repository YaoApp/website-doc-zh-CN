# 应用目录结构

YAO 应用目录结构说明

```
├─apis
├─data
├─db
├─flows
├─certs
├─imports
├─logs
├─models
├─plugins
├─schedules
├─scripts
├─stores
├─tables
├─forms
├─charts
├─tasks
|-langs
|-sockets
|-websockets
├─public
|-services
|-studio
├─widgets
│  └─dyform
└─icons

```

| 目录名称   | 功能介绍                           | 文件后缀        | 备注             |
| ---------- | ---------------------------------- | --------------- | ---------------- |
| apis       | 存放 API DSL 文件                  | `.http.yao`     |                  |
| data       | 应用数据目录                       | 多种文件格式    |                  |
| db         | 存放 SQLite 数据库文件             | `.db`           |                  |
| flows      | 存放数据流 DSL 文件                | `.flow.yao`     |                  |
| certs      | PEM 证书文件                       | `.pem`          |                  |
| imports    | 存放数据导入 DSL 文件              | `.imp.yao`      | 需要自己建文件夹 |
| logs       | 存放系统运行日志/调试信息          | `.log`          |                  |
| models     | 存放模型 DSL 文件                  | `menu.mod.yao`  |                  |
| plugins    | 存放 GRPC 插件扩展                 | `.so`           | 需要自己建文件夹 |
| schedules  | 存放定时任务/计划任务 DSL 文件     | `.sch.yao`      | 需要自己建文件夹 |
| scripts    | 存放 JavaScript 处理器             | `.js`           |                  |
| stores     | 存放 KEY-VALUE 存储器 DSL 文件     | `cache.lru.yao` | 需要自己建文件夹 |
| tables     | 存放表格 DSL 文件                  | `.tab.yao`      |                  |
| forms      | 存放表单 DSL 文件                  | `.form.yao`     |                  |
| charts     | 存放图表 DSL 文件                  | `.chart.yao`    |                  |
| tasks      | 存放延时任务/异步任务 DSL 文件     | `.task.yao`     | 需要自己建文件夹 |
| langs      | 存放应用语言包                     | `xxx.yml`       | 需要自己建文件夹 |
| sockets    | Socket 连接 DSL 文件               | `sock.yao`      | 需要自己建文件夹 |
| websockets | WebSocket 连接 DSL 文件            | `ws.yao`        | 需要自己建文件夹 |
| public     | 公开目录, 可作为静态资源服务器使用 | 多种文件格式    |                  |
| services   | 云函数脚本                         | `.js`           | 需要自己建文件夹 |
| studio     | Studio 脚本                        | `.js`           | 需要自己建文件夹 |
| widgets    | 存放自定义 DSL 脚本                | `.js`           | 需要自己建文件夹 |
| icons      | 存放系统图标                       |                 |                  |
